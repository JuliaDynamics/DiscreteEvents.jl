Examples

Examples

The examples are intended to show, how Simulate.jl can be used and how the different approaches to modeling and simulation can be effectively combined.

  • Two guys meet
  • Single server
  • Table tennis

Notebooks

  • greeting.ipynb: a first illustration
  • tabletennis.ipynb: a simple table tennis simulation.
  • singleserver.ipynb: single server modeling, simulation and analysis.
  • postoffice.ipynb: process based simulation of a post office,
  • dicegame.ipynb: process based simulation of Goldratt's dice game - must yet be adopted to Simulate!
